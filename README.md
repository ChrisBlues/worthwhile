# Worthwhile

✨ *Une application pensée comme un voyage intérieur, où chaque vidéo est une étape vers l’essentiel.*  

## Présentation
Worthwhile est une expérience immersive qui rassemble **six vidéos** soigneusement conçues, chacune inférieure à 50 Mo, pour rester fidèle aux contraintes de GitHub tout en préservant l’intégrité artistique du projet.  

## Contenu
- 6 vidéos intégrées, cœur vivant de l’application  
- Code HTML5/CSS3 clair et accessible  
- Structure harmonieuse, pensée comme une galerie numérique  

## Objectif
Offrir un espace contemplatif et moderne, où la technique se met au service de l’émotion. Worthwhile est une invitation à ralentir, à regarder, et à ressentir.
