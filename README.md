# worthwhile
30/10/2025 : Le point de départ est un tutoriel Youtube de Mr. Web Designer qui offre la possibilité d'utiliser cinq vidéos en page héro à l'aide de points de contrôle : cinq points sont visibles et chaque clic sur l'un d'eux fait apparaître une vidéo différente.
Le problème est que, si la démonstration sur le tuto est évidente, en local, les points de contrôle ne sont pas opérationnels et ne déclenchent pas les vidéos. 

31/10/2025 :
- Worthwhile est terminé. Il va être déposé sur Github distant dans sa version définitive (sauf corrections envisageables), avec les cinq vidéos de moins de 50 Mo (limite imposée par Github).
